# Privacy Policy — GPT Japanese Session Tutor

Effective date: 2026-04-15

Summary

This GPT (GPT Japanese Session Tutor) is a resource-only assistant that uses the static files included in this folder. We do not collect, store, or transmit user prompts, conversation transcripts, or any personally identifiable information as part of the packaged assistant.

What we do NOT collect

- User prompts or chat transcripts
- Account data, IP addresses, device identifiers
- Analytics, telemetry, or usage metrics

What the assistant uses

The assistant generates responses using only the static resources bundled in `knowledge_files/` (for example: `session_default.json`). These resources are part of the package and are the only data the assistant references by default.

Third-party integrations and deployments

If you deploy or integrate this assistant with hosted services, third-party APIs, or analytics tools, those services may collect data according to their own policies. This privacy policy applies only to the behavior of the packaged assistant as distributed in this repository.

Data retention

Because no user data is collected by this packaged assistant, there is no data retention related to user conversations. If logging or analytics are added in a deployment, document retention periods in your site-wide privacy policy.

Children

This assistant is not intended for use by children under the applicable age in your jurisdiction. Do not submit personal information about minors.

Security

Treat repository files as static resources. When deploying, follow standard security practices (access controls, secure storage of secrets) for any hosting environment or external service.

Contact

For questions about this assistant's privacy behavior, contact thru GitHub: [study_cards](https://github.com/BenMedeiros/study_cards)
